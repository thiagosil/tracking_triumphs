# README

This project is aimed to be centralize a bunch of things:

- Habit tracker
- Daily todo list
- Personal news

All those will be single user only. Thats why this project has very modest ambitions.


This app uses hotwire so we can use it as an app in the future.

The initial page will be a dashboard with all the main stuff we care about. A view with all the habits, a view with the daily todo list and a view with the personal news.

## Habit tracker

First thing will be a habit tracker. The idea is to have a list of habits that you want to track. You can mark them as done or not done. The idea is to have a calendar view where you can see how many times you did a habit in a month. The design for he calendar will be similar to the one they use on github.
